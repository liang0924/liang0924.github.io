二、基础语法

  • 注释：单行注释以#开头

​ 例： # 这是一个注释

多行注释可使用’’’或”””（注释内容放置其中）

​ 例：’’’ 这是一个注释 ‘’’；

  • 行与缩进

    Python中使用缩进来表示代码块，不需要使用大括号{}。缩进的空格数是可变的，但同一代码块的语句必须包含相同的缩进空格数。

  • 多行语句

可以使用反斜杠（\）来实现多行语句，但是在()、{}、[]中的多行语句不需要使用（\）

例如：total = item_one + \

​ item_two + \

​ item_three

total = [‘item_one’, ‘item_two’, ‘item_three’,

​ ‘item_four’, ‘item_five’]

  • 数字类型

一个有四种数字类型：整数，布尔型，浮点数和复数。

（只有一种整数类型Int，表示长整型）

  • 字符串

①单引号和双引号使用完全相同

②使用三引号可以指定一个多行字符串

③使用反斜杠可以转义，使用r可以让它 不发生转义。

④字符串可以用 + 运算符连接在一起，用 * 运算符重复。

⑤Python 中的字符串有两种索引方式，从左往右以 0 开始，从右往左以 -1 开始。

例如：str=hello print(str[0:-2]) #将输出hel

⑥Python中的字符串不能改变。Python 没有单独的字符类型，一个字符就是长度为 1 的字符串。

⑦可以在同一行中使用多条语句，语句之间使用分号(;)分割。

  • print默认输出是换行的，如果要实现不换行需要在变量末尾加上 end=””。

三、数据类型

变量不需要声明，每个变量在使用前都必须赋值，变量赋值后该变量才会被创建，在Python中，变量就是变量，它没有类型，所谓类型是变量所指的内存中对象的类型。

  • 多个变量赋值。

    可以同时为多个变量赋值，如：

​ a=b=c=1;

也可以为多个对象指定多个变量如：

a，b，c=1，2，”abc”

  • 标准数据类型

    不可变数据：Number（数字）、String（字符串）、Tuple（元组）；

    可变数据：List（列表）、Dictionary（字典）、Set（集合）。

  • Python3支持四种数字类型：int,float,bool,complex。内置的type()函数可以用来查询变量所指的对象类型。此外还可以用isinstance来判断，例如：

    type(a) #输出<class ‘int’>;

isinstance(a,int) #输出 True

二者区别在于type不会认为子类是一种父类类型，而isinstance会。

注意：在python2中是没有布尔型的，它用数字0表示false，用1表示true，到python3中将其定义为关键字，但它们的值还是0和1，可以与数字相加。

可以通过del语句删除一些对象引用，如：

del val1,val2,

  • 字符串

注意：与 C 字符串不同的是，Python 字符串不能被改变。向一个索引位置赋值，比如word[0] = ‘m’会导致错误。

  • List（列表）

列表是写在方括号 []之间、用逗号分隔开的元素列表。和字符串一样，列表可以被索引和截取。列表被截取后返回一个包含所需元素的新列表。截取方式与字符串相同，+、*所表示的含义也和字符串相同。

与字符串不同的是，列表中的元素是可以改变的。

截取语法格式：变量[头下标：尾下标：步长]

所谓步长是指间隔的位置

如果步长为负数表示逆向读取，例如：letters[-1::-1] #输出oikcehc

  • Tuple（元组）

元组与列表相似，不同之处在于元组的元素不能修改。元组写在小括号里，元素用逗号隔开，元组中的元素类型也可以不相同。

元组与字符串类似，可以被索引且下标索引从0开始，-1 为从末尾开始的位置。也可以进行截取（看上面，这里不再赘述）。其实，可以把字符串看作一种特殊的元组。

  • Set（集合）

集合是由一个或数个形态各异的大小整体组成的，构成集合的事物或对象称为元素或成员。基本功能是进行成员关系测试和删除重复元素。可以使用{}或set()函数创建集合。

注意：创建一个空集合必须使用set()而不是{}，因为{}是用于创建一个空字典。

创建格式：

parame={value01,value02,…}

set(value)

输出集合时重复的元素会被自动去掉

Set可以进行集合运算，如交并差等。

  • Dictionary(字典)

列表是有序的对象集合，字典是无序的对象集合。两者的区别在于：字典中的元素是通过键来存取的，而不是通过偏移存取。字典是一种映射类型，字典用{}标识，它是一个无序的键（key）：值（value）的集合。键必须使用不可变类型，在同一个字典中键的值唯一。

1、字典是一种映射类型，它的元素是键值对

2、字典的关键字必须为不可变类型，且不能重复

3、创建空字典是用{}.

总结：①符号：列表用[]，元组用()，集合和字典用{}。

​ ②字符串，列表，元组都可以进行截取，方法一致。元组中的元素类型可以不同。

四、详解数据类型

1、关于/和//：/是真除法，和平时使用的除法相同，//是向下取整除法，不过不一定得到的是整数，如果分子或者分母中含有小数，则结果为x.0。例如：15//3.11 得到4.0。

2、不同类型的数混合运算是会将整数转化成浮点数。

3、关于python字符串格式化，可使用f-string，称之为字面量格式化字符串。f-string 格式话字符串以 f 开头，后面跟着字符串，字符串中的表达式用大括号 {} 包起来，它会将变量或表达式计算后的值替换进去，实例如下：

4、列表的数据项不需要相同的类型。

5、元组和列表类似，不同之处在于元组的元素不能改变，创建元组只需要在括号中添加元素并用逗号隔开即可，不加括号也可以创建元组。

元组如果只包含一个元素，需要在元素后面添加逗号，否则括号会被当作运算符。虽然元组的元素值不允许修改，不过可以对元组进行连接组合。元组的元素值是不允许删除的，可以删除整个元组。

关于元组的不可变指的是元素所指的内存中的内容是不可变的。

6、字典：

del dict[‘Name’] #删除键‘Name’

dict.clear() #清空字典

del dict #删除字典

字典键的特性：①不允许同一个键出现两次。创建时如果同一个键被赋值两次，后一个值会被记住。

​ ②键必须不可变，所以可以用数字，字符串或元组充当，而用列表就不行。

7、集合（set）是一个无序的不重复元素序列

可以使用大括号 { }或者 set()函数创建集合，注意：创建一个空集合必须用 set() 而不是 {}，因为 {}是用来创建一个空字典。

①添加元素：s.add(x) #将元素x添加到集合s中，如果元素已存在则不进行任何操作。

​ s.update(x) #也可以添加元素，x可以是列表，元组，字典等，x可以有多个，逗号隔开。

thisset = set((“Google”, “Runoob”, “Taobao”)) #创建集合

thisset.add(“Facebook”)#添加元素

thisset.update({1,3})#添加元素，添加一个集合

thisset.update([1,4],[5,6])#添加元素，添加两个列表。

②移除元素：s.remove(x) #将元素移除，如果元素不存在则会发生错误

​ s.discard(x) #将元素移除，元素不存在不会发生错误

​ s.pop() #随机删除集合中的一个元素，函数返回值是某个元素。

六、条件和循环

  • 复合赋值

    a,b=0,1

  • end关键字

关键字end可以用于将结果输出到同一行，或者在输出的末尾添加不同的字符。

print(b,end=’,’) #输出不换行

  • if语句

Python 中用 elif代替了 else if，所以if语句的关键字为：if – elif – else。

注意：

①每个条件后面要使用冒号：，表示接下来是满足条件后要执行的语句块。

②使用缩进来划分语句块，相同缩进数的语句在一起组成一个语句块。

③在Python中没有switch – case语句。

  • while语句

    while 判断条件(condition):

注意冒号和缩进，在python中没有do..while循环

如果while循环体中只有一条语句，可以将该语句与while写在同一行中。

  • break语句和continue语句

break 语句可以跳出 for 和 while 的循环体。如果你从 for 或 while 循环中终止，任何对应的循环 else 块将不执行。

continue 语句被用来告诉 Python 跳过当前循环块中的剩余语句，然后继续进行下一轮循环。

注意：循环语句可以有 else 子句，它在穷尽列表(以for循环)或条件变为 false (以while循环)导致循环终止时被执行，但循环被 break 终止时不执行。

pass语句是空语句，是为了保持程序结构的完整性。pass不做任何事情，作为占位语句。
